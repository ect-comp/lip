Selecionando Ingredientes

Escreva um programa que lê um valor inteiro N e em seguida
lê os nomes de N ingredientes que formarão uma lista L1.
Depois, seu programa deve ler dois valores inteiros, X e Y.

O seu programa deve ordenar a lista de ingredientes L1
e então criar uma lista L2 com todos os ingredientes entre
os índices X e Y.

Ao final, seu programa deve imprimir a lista L2.

Você pode assumir que X e Y são índices válidos da lista L1.


- Exemplo de Entrada 1:
5
queijo
milho
tomate
presunto
ervilha
1 
3

- Exemplo de Saída 1:
\['milho', 'presunto', 'queijo'\]
